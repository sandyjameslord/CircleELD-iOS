# CircleELD - iOS version

Welcome to CircleELD for iOS! This is the first iOS version of the Circle-ELD curriculum for English Language Learners. 
Please run this in Xcode, and build using iPhone 11 or later.

Built using no 3rd party libraries.

Please let me know what you think!
